Installing Requirements
	
	Pip sudo apt-get install python-pip
	Flask sudo pip install flask
	ino sudo pip install ino
	
	Running the webpage:
		cd
		cd YRSWeather
		sudo python StartServer.py
	
	Open Manually:
		cd 
		cd YRSWeather
		cd ArduinoCode/Open
		sudo ino upload
	
	Close manually:
		cd
		cd YRSWeather
		cd ArduinoCode/Close
		sudo ino upload

Thats all
